Privacy Policy

The NoteD do not re-distribute or sell personal information collected on our web servers.

Information collected:

NoteD servers collect the following information:

  • Internet Protocol (IP) address of computer being used
  • web pages requested
  • referring web page
  • browser used
  • date and time

NoteD does not associate this data to individual user identities. In addition to the IP address/browser information collected from anonymous visitors to the site, other information is collected and stored when users log in to view restricted content or submit forms. Specifically, the following information is collected and mapped to a locally created account:

  • Email address
  • Username
  • First name
  • Last name
  • Date and time when joined NoteD
  • Date and time last log in

Additional information if you have signed up via Yandex:

  • Uid/id - the unique identifier of the Yandex user.
  • Login - User login on Yandex.
  • Client id - the ID of the OAuth application for which the OAuth token passed in the request was issued.
  • Display name - the name that is displayed for this account in the Yandex interface.
  • Real name - first and last name of the user specified by him in Yandex ID.
  • First name - the username specified by him in Yandex ID.
  • Last name - last name of the user specified by him in Yandex ID.
  • Sex - user Gender.
  • Default email / emails - the default email address for contacting the user.
  • Psuid - ID of an authorized user in Yandex. It is formed on the Yandex side based on the client_id and user_id pair.

Additional information if you have signed up via GitHub:

  • Login
  • Id
  • Node_id
  • Avatar_url
  • Gravatar_id
  • Open GitHub User URLs: profile, followers, following, gists, starred, subscriptions, organizations, repos, events, received events.
  • User type
  • Site_admin availability
  • Name
  • Company
  • Blog
  • GitHub Location
  • Email
  • Hireable
  • Bio
  • Twitter username
  • Public repos number
  • Public gists number
  • Followers number
  • Following number
  • Created at
  • Updated at

Use of collected information:

NoteD will only use personal information collected from this site for the purpose of communication back to individuals who contact us via the site.

NoteD uses browser-IP-address information and anonymous-browser history to report information about site accesses and for profiling purposes. This information is used to improve Web presentation and utilization. IP address information may also be used for troubleshooting purposes.

The GitHub, Yandex information collected by NoteD is used to identify users accessing secure content, and to pre-fill forms with information about the user submitting the form.

Distribution of collected information:

  • NoteD will not disclose, without your consent, personal information collected about you.
  • NoteD will not distribute or sell personal information to third-party organizations.
